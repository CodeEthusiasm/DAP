Na diretoria watchFolderPorjecto está o algoritmo de construção do watch folder para o projeto.

Este algoritmo tem como finalidade ativar um algoritmo de imagem sempre que um video.m4v (do drone) é introduzido numa diretoria específica (MOVED_IN_TO).

Neste algoritmo também são dados outros exemplos como IN_MODIFY | IN_CREATE | IN_DELETE.
