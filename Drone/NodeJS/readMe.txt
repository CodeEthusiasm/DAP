

Para controlar o drone via laptop é necessário:


(1) - 	Instalar o nodeJS dependendo do sistema operativo, 
	pode ser feito em: https://nodejs.org/en/download/package-manager/

(2)-	Em seguida ir a: https://github.com/hybridgroup/node-bebop

(3) -	Fazer uso da biblioteca fornecida em (2), neste caso "node-bebop"
	sendo que é a mais recente e uma compilação das que já foram
	anteriormente criadas.

(4) - 	Para fazer uso da mesma biblioteca tem de se usar o comando: "npm install node-bebop".

(5) - 	Criar os ficheiros com extenção ".js" e fazer a programação em javascript das rotinas.

(6) - 	Por fim ao criar as proprias rotinas.js há que executar as mesmas no terminal
	dependendo do local onde se instalou a biblioteca e se criou a rotina.

Nota: "Atenção para testar todas estas rotinas em segurança deverá ser usado a APP
android fornecida pela Parrot como segurança caso os testes não corram como esperado.
